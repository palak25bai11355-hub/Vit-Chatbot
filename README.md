# Vit-Chatbot
The VIT Smart Chatbot is a simple Python-based chatbot that provides basic information about VIT, such as fees, courses, hostel, and placements. It interacts through a text interface and includes features like greetings, time display, and a basic calculator, making it beginner-friendly and easy to use.

The chatbot can respond to the following user inputs:

 Greetings → Say "hi", "hello", "hey"
 Name Input → "My name is ___"
 Fees Information → Type "fees" and choose category (1–4)
 Hostel Info → Type "hostel"
 Courses Offered → Type "course"
 Placement Info → Type "placement"
 Current Time → Type "time"
 Calculator → Type "calculate 5+3"
 Basic Conversation → "how are you"
 Exit Program → Type "exit"
